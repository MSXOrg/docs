# VS Code Extensions

Editor tooling that brings the MSX way of working directly into **VS Code** — meeting developers where they already are, so the *easy* path and the *right* path are the same one.

These extensions are authored in TypeScript and inherit the [Coding Standards](../Coding-Standards/index.md) and [Ways of Working](../Ways-of-Working/index.md) from this site.
